# legalize_site/templatetags

## Назначение
Глобальные template tags проекта.

## Состав
- `i18n_compat.py` — совместимость для legacy-тегов перевода (`blocktranslate`, `translate`) через алиасы к встроенным Django-тегам.

## Примечание
Тег-библиотека подключается в настройках шаблонов как builtin, чтобы старые шаблоны продолжали работать без массового рефакторинга.
