# submissions/templates

## Назначение
Шаблоны SSR для приложения `submissions`.

## Файлы
- `submissions/submission_list.html` — список оснований.
- `submissions/submission_detail.html` — карточка основания.
- `submissions/submission_form.html` — форма создания/редактирования основания.
- `submissions/document_form.html` — форма документа внутри основания.

## Примечание
Именование шаблонов соответствует путям views и url-именам приложения.
