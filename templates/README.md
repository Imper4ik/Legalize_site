# templates (root)

## Назначение
Корневой каталог шаблонов Django, подключённый через `TEMPLATES["DIRS"]`.

## Что здесь находится
- `base.html`, `403.html`, `404.html` — базовый каркас и страницы ошибок.
- `account/*`, `socialaccount/*` — шаблоны аутентификации (allauth).
- `admin/*` — кастомизация Django Admin.
- `clients/*` и `clients/partials/*` — общие/переиспользуемые фрагменты клиентского интерфейса.
- `includes/*` — проектные include-компоненты.

## Правило
Если шаблон относится только к одному приложению, храните его внутри `app/templates/<app>/...`.
Глобальные overrides — в этом каталоге.
