# clients/templatetags

## Назначение
Кастомные template tags/filters для UI приложения `clients`.

## Модули
- `form_filters.py` — фильтры для удобного рендера форм.
- `client_filters.py` — форматирование клиентских данных в шаблонах.
- `formatting.py` — дополнительные универсальные форматтеры.
- `document_tags.py` — helper-теги для документов и чеклистов.

## Принцип
Логику форматирования, которая повторяется в шаблонах, выносите в templatetags вместо дублирования HTML-выражений.
