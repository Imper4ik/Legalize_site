# clients/views

## Назначение
HTTP-обработчики приложения `clients`, разделённые по контекстам использования.

## Модули
- `base.py` — общие миксины/декораторы доступа для staff.
- `clients.py` — список/детали/CRUD клиентов, печатные формы и связанные действия.
- `documents.py` — добавление/удаление/подтверждение и верификация документов.
- `payments.py` — создание, редактирование, удаление платежей.
- `reminders.py` — списки напоминаний и действия по ним.
- `emails.py` — предпросмотр и отправка email.
- `metrics.py` — dashboard метрик.
- `admin_views.py` — служебные admin-представления.

## Подход
Views остаются «тонкими», а вычисления и бизнес-правила выносятся в `clients/services`.
