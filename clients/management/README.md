# clients/management

## Назначение
Пользовательские management-команды Django для обслуживания клиентских данных.

## Команды
- `update_reminders` — актуализация напоминаний и рассылка уведомлений.
- `normalize_document_names` — нормализация названий документов.
- `fix_document_names` — исправления legacy-названий/несоответствий.
- `compress_images` — массовая компрессия загруженных изображений.

## Запуск
```bash
python manage.py <command_name>
```

## Использование
Команды подходят для cron/worker-задач и ручных административных операций.
