# clients/services

## Назначение
Слой бизнес-логики для `clients`, переиспользуемый из views, команд и фоновых процессов.

## Модули
- `calculator.py` — расчёт финансовых требований, конвертация EUR/PLN, кеш курса НБП.
- `notifications.py` — отправка писем клиентам/сотрудникам, шаблоны писем, PDF-подтверждения и логирование отправок.
- `wezwanie_parser.py` — извлечение данных из Wezwanie (PDF/изображения, OCR, даты, номера дела, тип письма).
- `pricing.py` — получение стоимости услуги (БД + fallback).
- `responses.py` — HTTP-helpers (например, no-store заголовки).
- `image_compression.py` — компрессия изображений для загрузок.

## Правило
Весь код, который можно протестировать независимо от HTTP-контекста, размещайте здесь.
